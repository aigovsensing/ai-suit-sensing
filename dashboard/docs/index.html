핵심 기능 (Key Features)

단순한 데이터 나열이 아닌, 법적 통찰력을 제공하는 강력한 도구들을 확인해보세요.

Interactive Lineage

원고, 피고, 데이터셋 간의 복잡한 관계를 Cytoscape.js 기반의 고성능 리니지 그래프로 시각화합니다.

Gemini AI Analysis

Google Gemini AI를 연동하여 소송 데이터를 바탕으로 전문적인 법적 트렌드 월간 보고서를 자동 생성합니다.

Global Heatmap

대륙별 줌인 기능과 연동된 인터랙티브 히트맵을 통해 전 세계 소송 분포를 한눈에 파악합니다.

Multi-dim Filtering

피고, 학습 데이터 분야, 진행 상태 등 다차원 필터링을 통해 필요한 정보를 즉시 필터링합니다.

Real-time Tickers

최신 소송 현황과 주요 뉴스 정보를 실시간 티커를 통해 대시보드에 공급합니다.

Responsive Design

데스크탑부터 모바일 기기까지 모든 환경에서 최적화된 시각화 레이아웃을 제공합니다.

인터랙티브 리니지 분석

소송의 주체와 객체 간의 관계를 그래프로 풀어냅니다. 특정 기술이나 기업이 어떤 법적 분쟁의 중심에 있는지 즉각적으로 식별할 수 있습니다.

  • 다중 피고 및 원고 관계 추적
  • 데이터셋별 소송 집중도 분석
  • 그래프 중심성 기반 리스크 감지

다양한 통계 대시보드

축적된 소송 데이터를 청구 내용, 국가, 피고 기업별로 집계하여 시각화합니다. 트렌드 차트를 통해 소송의 증가 추이를 분석하세요.

  • 청구 내용별 파이 차트
  • 연도별/월별 소송 트렌드 분석
  • 실시간 상위 피고 기업 순위

정밀한 리스트 및 검색

수천 건의 소송 데이터를 빠르고 정확하게 검색합니다. 상세 모달을 통해 판결문 요약 및 관련 뉴스에 즉시 접근할 수 있습니다.

  • 고속 전문(Full-text) 검색 엔진
  • 진행 상태별 스마트 배지 시스템
  • CSV v2.1 규격 완벽 호환

Quick Start

Docker를 사용하여 단 몇 초 만에 대시보드를 실행해보세요.

# Clone the repository
git clone https://github.com/aigovsensing/ai-suit-dashboard.git
cd ai-suit-dashboard

# Run with Docker Compose
docker-compose -f docker/docker-compose.yml up -d

# Access the dashboard at http://localhost:8007
