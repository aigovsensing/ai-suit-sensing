API Guide

AI Litigation Dashboard에서 제공하는 REST API 엔드포인트에 대한 기술적인 세부 정보를 제공합니다.

1. 데이터 조회 엔드포인트

GET /api/files

./data 디렉토리에 있는 CSV 데이터셋 목록을 반환합니다. 최신순으로 정렬됩니다.

// Response Example
["aisuit_20260429_1700.csv", "aisuit_20260414.csv"]
GET /api/cases

소송 데이터를 가져오는 메인 엔드포인트입니다. 특정 CSV 파일을 로드하거나 실시간 API 데이터를 페치합니다.

Params: file_name (optional)

GET /api/version

현재 애플리케이션의 빌드 버전(Git Tag)과 커밋 정보를 반환합니다.

2. 분석 및 보고서 엔드포인트

GET /api/statistics

대시보드 시각화를 위한 집계된 통계 데이터를 반환합니다.

POST /api/report/generate

Gemini AI를 사용하여 전문적인 월간 동향 보고서를 생성합니다.

// Payload
{
  "type": "filing_date",
  "month": "2026-05"
}
POST /api/report/download

마크다운 형식의 보고서를 워드(.docx) 파일로 변환하여 다운로드합니다.

3. 정적 파일 경로

  • /css/*: 스타일시트
  • /js/*: 자바스크립트 파일
  • /img/*: SVG 지도 및 이미지
  • /manual/*: 시스템 매뉴얼 (MD)
