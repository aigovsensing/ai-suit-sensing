User Guide

Detailed features and premium utilization methods of the AI Litigation Dashboard.

App Goal: To track and visualize copyright and rights infringement lawsuits arising from AI companies training models using unauthorized data, providing real-time geographical and relational insights.

1. 🚀 Data Loading & Visualization (Initial Setup)

  • Dataset Selection: Select a specific dataset from the CSV Select dropdown in the header to analyze a particular point in time.
  • Refresh Visualization: Before manually adjusting filters, click the Update Map button to synchronize the map and statistics with the latest data.

2. 🗺 Interactive Map Navigation

  • Global Toggle: Instantly switch between USA (US) and Global (WORLD) maps via the REGION menu in the header.
  • High-Precision Zoom (Marquee Zoom):
    • Shift + Drag: Draw a rectangle over a specific area to zoom in precisely to that region.
    • Mouse Wheel: Smoothly zoom in and out centered on your cursor position.
  • Smart Tooltips: Hover over any region to immediately see its name and the number of active lawsuits filtered in real-time.

3. 🎯 Advanced Multi-dimensional Filtering

  • Defendant Filter: Select a specific company (e.g., OpenAI, Meta, Google) to focus on litigation involving that entity.
  • Theme Filter: Filter by the data type involved in the lawsuit (e.g., Music, Visual Art, Code, Book) to identify industry-specific trends.
  • Multi-Status Filter: Select multiple progress statuses from the status dropdown to reflect them on the heatmap.

4. 📈 Statistics & Trend Analysis

  • Litigation Trend: View monthly and yearly litigation frequency via the 'Litigation Trend' chart at the bottom of the [STATISTICS] menu.
  • Comprehensive Stats: Provides six core charts including claims, defendant market share, and court decisions.

5. 🔍 High-Performance Case Search

  • Integrated Search: Enter keywords in the sidebar search box to simultaneously search across case titles, plaintiffs, defendants, and regional codes.
  • Result Listing: Matching lawsuits are listed as elegant cards; click any card to view detailed information immediately.

6. 🔑 Key Status Definitions (Glossary)

  • Ready: Lawsuit filed, but detailed documents are not yet public.
  • Started: Official legal proceedings and discovery have commenced.
  • 1st/2nd Instance: Trial ongoing in District or Higher Courts.
  • Decision: Final judgment rendered by the presiding court.
  • Appeal: Additional legal disputes ongoing following an appeal.
  • Closed: All proceedings finalized through judgment, settlement, or withdrawal.

7. 🕸️ Lineage Graph Analysis

An advanced visualization tool to understand the network between complex legal parties.

  • Plaintiff/Defendant Tracking: Visually analyze the connections between parties.
  • Dataset Centric Analysis: Identify which lawsuits are linked to specific datasets.
  • Focus Mode: Select a target to filter the view to show only the direct litigation history and relationship network.
